# CoreInventory - Odoo Hackathon Project

CoreInventory is a modular Inventory Management System built using Odoo.

Features:
- Product Management
- Incoming Stock (Receipts)
- Delivery Orders
- Internal Transfers
- Inventory Adjustments
- Stock Ledger Tracking

Tech Stack:
- Odoo
- Python
- PostgreSQL

This project was developed as part of the Odoo Hackathon.
